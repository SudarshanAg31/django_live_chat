# 💬 Django Live Chat

A real-time live chat application built with **Django**, allowing users to communicate instantly through a clean and responsive interface.

---

## 🚀 Features

- 🔐 User Authentication (Login & Signup)
- 💬 Real-time Messaging
- 👥 One-to-One Chat
- 📱 Responsive UI
- 🗄️ Database Storage for Messages
- ⚡ Fast and Secure Backend with Django

---

## 🛠️ Tech Stack

- **Backend:** Django
- **Frontend:** HTML, CSS, JavaScript
- **Database:** SQLite
- **Language:** Python

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/SudarshanAg31/django_live_chat.git
```

### 2. Move to the Project Folder

```bash
cd django_live_chat
```

### 3. Create a Virtual Environment

Windows

```bash
python -m venv venv
venv\Scripts\activate
```

Linux / macOS

```bash
python3 -m venv venv
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Apply Migrations

```bash
python manage.py migrate
```

### 6. Run the Server

```bash
python manage.py runserver
```

Open your browser and visit:

```
http://127.0.0.1:8000/
```
